# yama-normalize
yamanoku.net default style

## Support Browser
- Chrome
- Firefox
- Safari
- Edge

## Install

```
$ yarn add yama-normalize
```

or

```
$ npm install yama-normalize
```

## Usage

```
@import 'node_modules/yama-normalize/yama-normalize.css';
```

```
<link rel="stylesheet" href="node_modules/yama-normalize/yama-normalize.css">
```
